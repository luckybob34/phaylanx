# Quanta Brand Guidelines Skill

## Overview
This skill applies Quanta Services' official brand guidelines to any artifact (presentations, documents, reports, web pages, etc.). It ensures consistent, professional brand expression across all materials.

## When to Use This Skill
Use this skill when creating or reviewing:
- PowerPoint presentations (.pptx)
- Word documents (.docx)  
- Excel spreadsheets (.xlsx)
- HTML/web pages
- PDFs and reports
- Marketing materials
- Internal communications
- External communications

## Core Brand Elements

### Logo Usage

**Logo Formats:**
- **Horizontal format (PREFERRED)**: Use by default for most applications
- **Vertical format**: Use when suited to layout composition
- **Icon only**: Use only where logo is too large/detailed (e.g., social media avatars)
- **NEVER use wordmark alone** - always pair with icon

**Logo Variants:**
- **Dimensional, Full Color (PREFERRED)**: Use wherever possible for depth and refinement
- **Flat, Full Color**: Use at small sizes (<70px or 0.625" height) or limited color processes
- **Flat, One Color**: Use for screen printing, embroidery, engraving

**Clear Space & Sizing:**
- Clear space: Use width of "QU" letters as minimum spacing
- Never let text, graphics, or other logos enter clear space
- **Minimum sizes:**
  - Dimensional: Digital 70px, Print 0.625"
  - Flat: Digital 50px, Print 0.375"

**Logo Misuses to AVOID:**
- ❌ Don't stretch, scale disproportionately, or distort
- ❌ Don't recreate logo or add text/graphics
- ❌ Don't scale, reposition, rotate, or flip icon/logotype
- ❌ Don't apply shadows, bevels, outlines, gradients, effects
- ❌ Don't place on low-contrast color or busy image
- ❌ Don't rotate, shear, or skew
- ❌ Don't recolor
- ❌ Don't alter transparency or use as watermark

### Typography

**Primary Typeface - Alternate Gothic Extra Condensed ATF:**
- **Uses:** Large headings (>30pt), short callouts (<175 chars), small labels (<30pt, <40 chars)
- **Weights:** Light, Regular, Demi
- **ALWAYS USE ALL CAPS**
- **Fallback for internal docs:** Arial in title case

**Secondary Typeface - Proxima Nova:**
- **Uses:** Body text, small headings (<30pt), captions, callouts
- **Weights:** Light, Regular, Italic, Bold
- **Can use:** Mixed case or all caps

**Typographic Hierarchy:**

1. **Large Headings (Display):**
   - Font: Alternate Gothic Extra Condensed ATF Demi
   - Case: ALL CAPS
   - Tracking: 100
   - Leading: ~108% font size

2. **Section Headings:**
   - Font: Alternate Gothic Extra Condensed ATF Demi
   - Case: ALL CAPS
   - Tracking: 200
   - Leading: ~130% font size

3. **Sub-headings (Labels):**
   - Font: Proxima Nova Bold
   - Case: ALL CAPS
   - Tracking: 200
   - No line breaks

4. **Body Headings:**
   - Font: Proxima Nova Bold
   - Case: Title case
   - Leading: ~145% font size

5. **Body Text:**
   - Font: Proxima Nova Regular
   - Leading: ~175% font size
   - Generous spacing between paragraphs

6. **Bullets:**
   - Character: Black Square glyph
   - Color: Infrared (light backgrounds) or Bolt (dark backgrounds)
   - Size: ~70% font size

**Heading Treatments (use sparingly, ~20% of headings):**
- **Alternating weights:** Demi for emphasis, Light for rest
- **Contrasting colors:** High contrast only on 1-2 key words
- **Outlines:** Only for short headlines at large sizes (72pt+), thin elegant stroke, align inside

### Color Palette

**Core Colors:**

**BOLT (Primary Brand Color - Orange)**
- HEX: #F0941C
- RGB: 240, 148, 28
- CMYK: 3, 49, 100, 0
- Pantone: 7564 C / 129 U

**CARBON (Primary Dark)**
- HEX: #221F1F
- RGB: 34, 31, 31
- CMYK: 0, 0, 0, 100
- Pantone: Black C / Black U

**INFRARED (Accent Red)**
- HEX: #CD0A1B
- RGB: 205, 10, 27
- CMYK: 13, 100, 100, 4
- Pantone: 186 C / 186 U

**Extended Palette:**
- Bolt: -1 (#D87C0A), Base (#F0941C), +1 (#FFAE4D)
- Carbon: -1 (#1A1717), Base (#221F1F), +1 through +8 (various grays), White (#FFFFFF)
- Infrared: -1 (#AA0011), Base (#CD0A1B), +1 (#EF1220)

**Color Usage:**
- **Primary combinations:** Bold Bolt or Infrared with Carbon or white
- **Accessibility:** Ensure WCAG contrast compliance
- **Avoid:** Combining Infrared with Bolt or medium Carbon tints

### Patterns

**Two Pattern Types:**

1. **Topographic (Bolder, More Active):**
   - Use for prominent features
   - Suggests contour lines, energy, movement
   - Rotation: Only in 90° increments

2. **Diagonal Hatched (Subtler):**
   - Use for subtle texture or small sizes
   - 45° diagonal lines
   - More refined expression

**Pattern Usage:**
- Add subtle texture and depth to layouts
- Don't place under small text (<24px digital, <14pt print)
- Can use dark pattern on mid-tone backgrounds for flexibility
- Available in all core colors

**Pattern + Logo:**
- Place logos on solid backgrounds when possible
- If using pattern with logo, ONLY use Carbon or Carbon +8 patterns
- Never use Bolt, Infrared, or Carbon +3 patterns with logo

**Pattern + Headings:**
- Place pattern under solid/demi weight text (not light/outlined)
- Keep outlined text on solid backgrounds

### Photography Guidelines

**Core Principles:**
- **Authentic & Spontaneous:** Real employees doing real work
- **No posed shots** (except specific employee highlights)
- **No stock photos** for external communications
- **Safety approved:** All field operations photos must be safety-approved
- **Watch for vulgarity:** Edit out offensive hard hat stickers/language

**Content Requirements:**
- **Represent entire workforce:** Show diversity in age, gender, race, occupation, ability
- **Breadth of services:** Electric, renewable, underground power, communications, storm response
- **Action & interest:** Energy, movement, compelling compositions, uncommon perspectives
- **All job roles:** Designing, installing, repairing, maintaining
- **Fresh & current:** Use photos from past ~5 years

**Technical Standards:**
- **Ambient lighting:** No harsh flash, subtle light modifiers okay
- **Crisp & bright:** Professional editing, avoid overprocessing
- **No cheesy effects:** No fake lens flares, excessive sharpening, over-saturation

**Sustainability Photography:**
- **Light, bright, clean, airy** aesthetics
- **Prioritize clean energy:** Solar, wind, battery, EV infrastructure
- **Minimize visible impact:** Avoid focal points on excavation, grading, access roads

### Sub-Branding & Co-Branding

**Sub-Branding (Quanta-Operated Companies):**
- Format: Quanta icon + company name in Alternate Gothic
- Use same brand system (colors, fonts, patterns)
- Companies WITHOUT "Quanta" in name should also add co-brand

**Co-Branding (Independently Operated):**
- Text: "A QUANTA SERVICES COMPANY"
- Placement: Below operating company logo
- Spacing: 3× height of capital "A"
- Alignment: Match operating company logo alignment
- Size: Smaller than logo taglines/sub-text
- Special icon version: No gaps around bolt tip
- Use for all external communications

## Implementation Guidelines

### For Presentations (PPTX):
1. **Cover slide:** Large Alternate Gothic heading with pattern background
2. **Title slides:** Alternate Gothic headings, Proxima Nova body
3. **Content slides:** Clear hierarchy with generous white space
4. **Color scheme:** Primarily Carbon/White with Bolt or Infrared accents
5. **Logo:** Dimensional full-color in footer or corner with clear space
6. **Photos:** Full-bleed when possible, authentic action shots

### For Documents (DOCX):
1. **Headers:** Alternate Gothic Extra Condensed ATF Demi, ALL CAPS
2. **Body:** Proxima Nova Regular, 175% leading
3. **Headings:** Proxima Nova Bold for sub-heads
4. **Margins:** Generous spacing, sophisticated airy appearance
5. **Color:** Primarily black/gray with Infrared or Bolt for emphasis
6. **Lists:** Black square bullets at 70% font size

### For Web/HTML:
1. **Hero sections:** Large Alternate Gothic headings with background patterns
2. **Navigation:** Proxima Nova Bold, clear hierarchy
3. **Body text:** Proxima Nova Regular, excellent readability
4. **CTAs:** Infrared buttons with white text, Infrared "READ MORE" arrows
5. **Contrast:** WCAG AA compliant minimum
6. **Responsive:** Maintain clear space around logo at all sizes

### For Spreadsheets (XLSX):
1. **Headers:** Proxima Nova Bold, Infrared or Bolt background
2. **Data:** Proxima Nova Regular, professional appearance
3. **Charts:** Use brand color palette
4. **Logo:** Top corner with appropriate clear space

## Key Brand Personality Traits

- **Bold:** Confident use of color and typography
- **Exciting:** Dynamic patterns, action photography, energy
- **Confident:** Strong hierarchies, assertive design choices
- **Warm:** Human-focused photography, accessible language
- **Professional:** Sophisticated spacing, high-quality materials
- **Powerful:** Literal and metaphorical power representation

## Common Mistakes to Avoid

1. **Typography:**
   - ❌ Using Alternate Gothic in mixed case
   - ❌ Too-tight leading on headings
   - ❌ Not enough spacing between paragraphs
   - ❌ Using wordmark without icon

2. **Color:**
   - ❌ Infrared + Bolt combinations (contrast issues)
   - ❌ Not meeting WCAG contrast requirements
   - ❌ Recoloring the logo

3. **Layout:**
   - ❌ Insufficient clear space around logo
   - ❌ Pattern under small text
   - ❌ Busy backgrounds behind logo
   - ❌ Not enough white space

4. **Photography:**
   - ❌ Using stock images for external materials
   - ❌ Posed, unnatural shots
   - ❌ Lack of diversity representation
   - ❌ Non-safety-approved field photos

## Web Presentation Template

A reusable HTML slide deck template is available at `docs/.formatting/presentation-template.html`. Use it as the starting point whenever creating a new fullscreen web presentation with QUANTA branding.

### Template File
`presentation-template.html` - ~6 placeholder slides with all brand CSS, sidebar nav, and JS engine included.

### Slide Types

| Class | Background | Use For |
|---|---|---|
| `slide-dark` | Carbon (#221F1F) + diagonal grid | Hero/title, stats, key metrics |
| `slide-light` | White | Tables, dense content, runbooks |
| `slide-gray` | #F4F4F4 | Two-column layouts, supporting slides |
| `slide-accent` | Carbon gradient + bolt overlay | Section divider transitions |

### Component Classes

Copy any of these into `.slide-inner` as needed:

| Class | Purpose |
|---|---|
| `.two-col` | Side-by-side grid layout |
| `.data-table` | Full-width bordered table with bolt header |
| `.stat-grid` / `.stat-item` / `.stat-value` | Large-number KPI metric grid |
| `.highlight-box` / `.highlight-grid` | Bolt-accented callout fact boxes |
| `.callout` | Bold left-border quote or note |
| `.card-grid` / `.card` | Auto-fit card grid |
| `.program-element-list` | Icon-prefixed feature/principle list |
| `.pillar-grid` / `.pillar` | Icon + label + text pillar columns |
| `.badge` (+ `.badge-bolt`, `.badge-infrared`, etc.) | Small inline label chips |
| `.runbook-phases` / `.runbook-phase` | Operational step phase cards |
| `.code-block` | Dark monospace code snippet |

### Nav & JS Conventions

- Each `.nav-link` that navigates must have `data-slide="N"` (0-indexed).
- Section labels that are not clickable use class `.nav-link.nav-divider`.
- `const TOTAL_SLIDES = N` at the top of the `<script>` block must match the actual slide count.
- `goToSlide(index)` is the only navigation function - call it from any click handler.
- Keyboard: Arrow keys, Space/PageDown, Home/End all work automatically.
- Touch swipe threshold: 50px. Wheel debounce: 50ms.

### Creating a New Presentation

1. Copy `presentation-template.html` to your project's `docs/presentation/index.html`.
2. Update `<title>`, `.nav-brand`, and `TOTAL_SLIDES`.
3. Replace placeholder slides - keep `data-index` values sequential from 0.
4. Update `.nav-list` items - one `.nav-link[data-slide]` per navigable slide.
5. Update `progress-fill` initial width: `(1 / TOTAL_SLIDES * 100)%`.

## Usage Notes

When applying this skill, Claude should:
1. Analyze the artifact type and purpose
2. Apply appropriate typography hierarchy
3. Use brand colors correctly with proper contrast
4. Ensure logo usage follows all guidelines
5. Create sophisticated, airy layouts with generous spacing
6. Recommend authentic photography approaches
7. Flag any brand guideline violations
8. Offer alternatives that maintain brand consistency

This skill ensures every artifact powerfully represents the Quanta brand while maintaining accessibility, professionalism, and visual impact.
